## [Play Game](https://snac-man.vercel.app)
SNAC-MAN is a hybrid game based on Snake and PAC-MAN where you run around a maze, eating food and avoiding ghosts to grow longer and get a highscore!

### Tech Stack
- NextJS
- TypeScript
- JavaScript
- Express
- Tailwind
- MongoDB
- React
- Deployed on Vercel
